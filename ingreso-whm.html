Cómo ingresar al WHM

El WHM (Web Host Manager) es un panel de control que te permite administrar las cuentas de hosting de tu Plan Reseller o Revendedor. Las opciones son muchas, dentro de las cuales las más importantes son:

  • List Accounts: Vas a poder ver listadas todos los dominios creados hasta el momento. Desde allí podrás cambiarles la contraseña haciendo click en el signo “+” que tiene cada dominio del lado izquierdo. También te muestra los nombres de usuario, el espacio utilizado y el asignado para cada cuenta en particular.

  • View Bandwidth Usage: Podrás ver estadísticas de la transferencia utilizada por cada dominio creado.

  • Create a New Account: Desde aquí podrás crear nuevos dominios.

  • Limit Bandwidth Usage: Te permite ver la tranferencia utilizada por cada dominio y modificarla individualmente según las necesidades.

  • Password Modification: Podrás cambiar la contraseña de cada cuenta creada.

  • Quota Modification: Te permite ver el espacio utilizado por cada dominio y modificarlo individualmente según las necesidades.

  • Suspend/Unsuspend an Account: Desde aquí podrás suspender o dessuspender cuentas creadas, la información subida al sitio no se borrará ni tampoco las cuentas de mail creadas o los mails que contengan.

  • Terminate an Account: Te permite eliminar cuentas totalmente.

  • Upgrade/Downgrade an Account: Para cambiar el plan de cada una de las cuentas creadas.

  • AddPackages: Podrás crear nuevos planes de hosting para asignarles a las cuentas que crees.

Con esta guía podrás aprender cómo ingresar al WHM para luego administrar los distintos dominios.

Los pasos son los siguientes:

Debes ingresar en el navegador, ya sea Internet Explorer, Mozilla Firefox, o el que utilices, y tipear: http://tudominio.com/whm

Vas a tener que reemplazar “tudominio.com” por el nombre de dominio de tu sitio web.

    01

    Escribí el usuario y contraseña de que te fueron asignados por Neolo.com en el mail de alta (no confundir con el nº de cliente y contraseña asignados para el panel de facturación de Neolo.com) y clic en “Aceptar”.

Has ingresado al WHM correctamente. Felicitaciones!
